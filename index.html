SystemPropertiesChecker

Some little tool I wrote to get quick information about installed service packs on windows.
Since Windows 7 SP1 was the last one of those service packs, I decided to extend the functionality to display installed .Net frameworks and other software.
Download as .zip Download as .tar.gz Repository GitHub Account
Currently implemented features
  • Gives information regarding device name, current IP, and current Windows information. It also shows information regarding a domain connected
  • Lists former installed Windows versions / builds
  • Lists .net SDKs, runtimes and frameworks installed


  • Lists installed browsers and some other tools, such as Git for Windows or PowerShell (Core is not implemented)
